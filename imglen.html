imglen

you can use it same as ImageLens. if this img element initialized, even though used the selector incloud this, it will be skip.

Demo 1 - Default lens

$("#img_01").imglen();

 

Demo 2 - Custom lens size

$("#img_02").imglen({ lensSize: 200 });

 

Demo 3 - Custom image source

$("#img_03").imglen({ imageSrc: "sample01.jpg" });

 

Demo 4 - Other lens properties

$("#img_04").imglen({ borderSize: 8, borderColor: "#06f" });

 

Demo 5 - Create image source 

$("p.create").on("mousemove", "img", function() {$(this).imglen({borderColor: "green"});});

 

Demo 6 - class.all use option 

		$("img.all").imglen({borderColor: "red" });

 

Demo 7 - class.all use option 

		$("img.all").imglen({borderColor: "red" });

 

Demo 8- add class imglen

when your img element add class imglen,
when mousemove this will become a imglen and has
the default style.
if you init it before mousemove, it will apply your option.
